Get Started

Install in 3 Steps

Get up and running in under a minute.

1

Clone or Download

Clone the repository from GitHub or download the ZIP file.

2

Load in Chrome

Go to chrome://extensions, enable Developer Mode, and click "Load unpacked".

3

Get API Key

Create a free API key at the Klipy Partner Panel and set it in the extension settings.
